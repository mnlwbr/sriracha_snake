# stationary_sriracha_snake
a snake game written in python

## Installation

```
pip install -r requirements.txt
```

in case of windows

```
pip install -r windows-requirements.txt
```

## Usage
Run the game
```
python spicy_snake/snake.py

```

## Run the tests

Run the tests with

	pytest (-v)

or calculate test coverage with:

	pytest --cov
